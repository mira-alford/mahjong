# MAHJONG TO THE DEATH

<img width="278" height="263" alt="mahjong causes great damage" src="https://github.com/user-attachments/assets/d93affd7-3080-4f4b-8994-20980614961b" />

A roguelike featuring MAHJONG battles

If you want to run this game, install [rust](https://rust-lang.org) and run `cargo run` in the repository root directory. Don't worry if it takes >10 minutes to compile, that's normal ;)

The mahjong tile sprites are from [here](https://github.com/FluffyStuff/riichi-mahjong-tiles). The music is by me (villuna)
